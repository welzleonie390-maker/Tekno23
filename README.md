# Tekno23 Ultimate

Ein Termux-Projekt für:

- Sample-Verwaltung
- BPM-Erkennung
- Korg Electribe Sampler ES2 Presets
- Datenbank
- Export
- Downloader
- Logger
- Debugger
- Backup
- Plugin-System

## Installation

```bash
chmod +x install.sh
./install.sh
```

## Start

```bash
tekno23
```

## Ordner

- samples/
- presets/
- exports/
- downloads/
- backup/
- logs/
- plugins/
